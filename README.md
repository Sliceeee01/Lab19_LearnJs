## Описание (что изучили)

В ходе лабораторной работы я познакомился с основами языка программирования JavaScript:

- Циклы: `for`, `while`, `do...while`, `for...of`
- Операторы `break` и `continue`
- Вложенные циклы
- Функции: классические, стрелочные, функции как значения, замыкания
- Массивы: создание, доступ к элементам, добавление/удаление, перебор, поиск элементов
- Объекты: создание, свойства, методы, добавление и удаление свойств, перебор свойств через `for...in`
- Отличия JavaScript от C#
